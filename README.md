## в—Ҳ What Is This
 
**PixelQR** - Interactive 8x8 Pixel-Art Canvas, lives entirely inside a QR Code.


Not *linked from* a QR code. Not *hosted somewhere* a QR code points to.  
**Inside the QR code.** The program IS the QR code. Scan it. The app opens. There is no server. There is no URL. There is no cloud. There is no `npm install`. There is only the void вҖ” and a glowing green grid.

## в—Ҳ Built for [HackClub SayCheese](https://saycheese.hackclub.com) рҹ§Җ
 
> *"fit it in a QR code вҶ’ profit???"*  вҖ” the SayCheese manifesto
 
SayCheese is a Hack Club initiative with a single, absurd, beautiful rule: **your entire program must fit inside a QR code.** No asterisks. No "the QR links to a CDN." The program. In the code. Scannable. Runnable. Real.

---


### в—Ҳ Features (The Chaos Manifest)
 
#### What It Does
 
- **Draw & Erase** вҖ” Click or drag across the 8Г—8 grid to toggle pixels green/dark
- **Symmetry Modes** вҖ” Cycle through `SYM вҶ’ H вҶ’ V вҶ’ Q` for horizontal, vertical, and quad-radial mirroring
- **Invert** вҖ” Double-click the canvas, press `i`, or enter the Konami Code to flip all pixels
- **Transient by Design** вҖ” Zero persistence. No `localStorage`. No cookies. No sync. Your art lives for exactly as long as the tab is open. Impermanence is the point.
 
#### What It Deliberately Does Not Do
 
- вқҢ Save your work
- вқҢ Export your creation
- вқҢ Phone home to analytics
- вқҢ Ask for notification permissions
- вқҢ Load a 47 kB icon font to render one button
- вқҢ Import React
- вқҢ Have a loading spinner (it loads in 0ms)
- вқҢ Require a login
- вқҢ Have a cookie consent banner
- вқҢ Exist on any server
